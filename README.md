# BIIA

---

## Desarrollo local

1. Instalación de todos los packages

```
yarn install
```

2. Inicialización del server local
```
yarn serve
```

3. Abrir la dirección que muestra la terminal para ver el sitio
4. Una vez realizados los cambios, se podrán subir los cambios hacia el repositorio


---

## Deployment a Production

El repositorio está conectado a Netlify para publicar los cambios directamente a producción una vez que se sube código al repositorio en la rama master (la default).

No se necesita hacer ninguna configuración extra para subir cambios con el estado actual.
